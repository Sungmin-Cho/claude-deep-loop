## Findings

### 🔴 Red — Blockers

1. **Same-ID review dispatch re-derives mutable inputs instead of replaying / 2-of-2**
   - **Sources**: Codex standard P2; Codex adversarial Red.
   - **Evidence**: the first verified read finds and authenticates a checker by
     `dispatch_request_id_digest`, but execution continues into reviewer resolution, hill-climb
     contract-file validation, and `latestInsights`. The later `newEpisode` recovery compares those
     newly derived values with the already committed episode. A response-loss retry can therefore
     fail when the contract or latest insights change after commit, despite using the exact same
     request ID. The only exact retry fixture disables hill climbing.
   - **Required response**: after full current fence/business-policy and request-digest validation,
     reconstruct the bounded response solely from the durable checker projection before any mutable
     filesystem/evidence derivation. Add hill-climb retry tests that change the latest insights and
     contract availability after the original commit.

2. **Episode creation proof is optional when its discriminator is removed / solo 1-of-2**
   - **Source**: Codex adversarial Red; independently reproduced from
     `verifyEpisodeCreationCorrelation`.
   - **Evidence**: both creation events and episode objects are filtered to
     `creation_contract === 'episode-create-v1'`. A hash-valid rewrite can remove that field from an
     existing checker, or inject an unversioned episode, and the verifier ignores the object instead
     of requiring a creation proof. Existing corruption tests retain the discriminator.
   - **Required response**: define a bounded legacy baseline from the initialization/migration
     boundary, require every post-baseline episode and `episode-new` event to use the v1 contract,
     reject discriminator downgrade and unversioned injection, and test both state-only and paired
     state/event rewrites.

     symlinks, require an `EEXIST` winner to be a regular non-symlink file whose real parent remains
